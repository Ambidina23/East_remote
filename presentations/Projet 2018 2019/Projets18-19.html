Généralités
  • Ces projets ( labelisés M2I* ) vont vous permettre de créer ou de modifier des logiciels libres le plus souvent pour des applications à grande distribution
  • Certains de ces projets sont en continuité avec ceux réalisés les années précédentes
  • Ils sont réalisés
    • en binômes .... ou plus après accord
    • quand vous le voulez avec des points de rencontre à l'Université aux heures prévues dans l'emploi du temps
  • Tous les développements effectués dans le cadre de ces projets
    • sont mis en Open Source sous licence CC BY-NC-SA par défaut ou autre licence libre selon les cas
    • sont placés ainsi que toute la documentation de développement et utilisateur sur un site git public

Présentation des projets M2 MIAGE INI 6 Novembre 2018

Projets autour de EAST
  • EAST : EAsy Slide Toolkit
  • EAST est un logiciel de présentation Open Source ( équivalent à Powerpoint ou Libreoffice Impress )
  • Il permet de faire des présentations en HTML visualisables sur n'importe que navigateur Web
  • Il s'appuie sur un langage XML dont les utilisateurs fabriquent des instances soit à la main soit à l'aide d'outils de saisie
Présentation des projets M2 MIAGE INI 3/8 6 Novembre 2018

Projet M2I-1 : Convertisseurs entre EAST et les autres logiciels de présentation
  • Tous les formats actuels de bureautique sont en fait une archive comprimée de fichiers XML
    • .odp pour Libre Office Impress
    • .pptx pour Microsoft Powerpoint
  • En s'appuyant sur les outils XML, l'objectif du projet est de réaliser tous les convertisseurs entre EAST, Powerpoint et Libreoffice Impress
    • Repérage des structures des différents formats
    • Conversion au niveau XML par XSLT

Projet M2I-2 : Elargissement du système de session de EAST

Etat d'avancement

  • EAST ( EAsy Slides Toolkit) est un logiciel de présentation en HTML analogue à Powerpoint
  • L'ensemble des actions effectués sur EAST est mémorisé avec le timing dans un fichier XML que EAST peut rejouer automatiquement

Reste à faire

  • Corriger un petit problème de fonctionnement sur les navigateurs récents
  • Ajouter l'audio au système : enregistrement dans le fichier XML et rejouer
  • Etudier la possibilité de créer une scène SMIL Timesheets automatiquement à partir du fichier XML

Projet M2I-3 : EAST Remote Control

Objectifs du projet

  • Permettre le pilotage des présentations EAST à distance
  • Mettre en place une application Web
    • qui s'affiche sur un dispositif distant : smartphone, tablette , ..
    • qui communique avec un serveur node.js hébergé sur la même machine que la présentation EAST
  • Mettre en place un serveur node.js qui va dialoguer avec l'application Web et avec EAST
    • pour recevoir les commandes de l'application Web
    • pour piloter EAST sur le serveur

Projet M2I-4 : Interface Web de saisie des présentations EAST

Objectifs du projet

  • Fournir une interface Web pour faciliter la saisie des présentations EAST
  • La création d'une présentation EAST passe par la création du fichier XML qui en constitue le coeur
  • Celle -ci peut se faire
    • à la main avec un éditeur de texte : long et source d'erreurs
    • avec un éditeur XML : plus facile et sécurisé par le schéma XML
  • L'objectif est de créer une interface Web de saisie qui alliera les deux : facilité et sécurite
    • Vérification en temps réel du schéma XML
    • Interface graphique pour la saisie des différentes structures XML
    • ATTENTION : il ne s'agit pas de faire une interface WYSIWYG

Présentation des projets M2 MIAGE INI 8/8 6 Novembre 2018

Problème

    Francais

  • La configuration de EAST n'a pas été trouvée et cela empêche le fonctionnement normal de EAST.
  • Pour corriger ce probleme, placez le répertoire config_EAST à côté de votre fichier HTML dans le même répertoire et rechargez votre fichier HTML

    English

  • EAST configuration has not been found and EAST cannot work normally.
  • To fix this problem, just put the directory config_EAST beside your HTML file in the same directory and reload your HTML file
